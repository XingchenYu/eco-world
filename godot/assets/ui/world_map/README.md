# World Map Base Texture

`arelian_world_map.png` is a project-local 2D world map base texture.

Source data:
- Natural Earth 1:50m shaded relief raster (`SR_50M`), public domain.
- Source page: https://www.naturalearthdata.com/about/terms-of-use/

Processing:
- Cropped to an East Africa and western Indian Ocean relief region.
- Recolored into a game-ready terrain/ocean base map.
- No roads, labels, route lines, cards, or network overlays are baked into the texture.
