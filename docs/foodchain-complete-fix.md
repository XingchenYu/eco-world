### 3. 食物因子机制（负反馈）

**原理**：一个物种过多 → 吃掉更多食物 → 食物减少 → 该物种数量下降

**实现**：
```python
# 食草动物
food_factor = 食物数量 / (同类数量 × 理想比例)

# 捕食者
food_factor = 猎物数量 / (同类数量 × 理想比例)

# 天敌压力
predator_pressure = max(0.3, 1.0 - 天敌数量 × 0.05)

# 最终繁殖率
有效繁殖率 = 基础繁殖率 × food_factor × predator_pressure
```

### 6. 生态平衡机制

```
          食物链关系图
          
【水生】
藻类 → 虾/浮游生物 → 小鱼 → 鲶鱼/大鱼
                    ↓
                  鲤鱼 → 黑鱼/狗鱼
          
【陆地】
草 → 兔子/鹿 → 狼
  → 昆虫 → 蜘蛛/鸟/青蛙
        → 狐狸/蛇
```

**平衡逻辑**：
1. **猎物过多** → 捕食者食物充足 → 捕食者繁殖增加 → 猎物减少
2. **猎物过少** → 捕食者食物不足 → 捕食者繁殖减少/饥饿死亡 → 猎物恢复
3. **物种泛滥** → 吃光食物 → 食物减少 → 该物种繁殖下降/饥饿死亡
