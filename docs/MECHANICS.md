# EcoWorld 机制说明

## 生命周期

所有生物共享这些基础机制：

- 年龄增长
- 饥饿累积
- 健康下降
- 死亡
- 进食回血
- 动态繁殖率

基础实现位于 [src/core/creature.py](../src/core/creature.py)。

## 动态繁殖

项目当前不采用“固定繁殖 + 固定数量上限”的简单模式，而是综合：

- 食物可用性
- 天敌压力
- 当前健康
- 当前饥饿
- 软承载抑制

## 多样化食谱

当前版本不再把大多数物种限制为单一食物源，而是采用三层食谱：

- 主食：优先寻找的稳定食物
- 替代食物：主食不足时可切换的次优来源
- 机会型猎物：靠近时会顺手捕食或啃食的资源

这套设计有两个目标：

- 减少单一物种暴跌后整条链路同步饿死
- 让水边、林缘和湿地出现跨生态位取食行为

实现上，繁殖判断和觅食判断被刻意分离：

- 觅食可以更宽，允许生物活下来
- 繁殖仍然主要看核心食物和天敌压力，避免因为“什么都能吃一点”而爆量

## 自然迁入

当前版本加入了低密度关键物种的自然迁入机制，但它不是“凭空刷新”：

- 来源位置必须符合地理约束
- 陆地物种从地图边缘陆地进入，模拟外部生态廊道
- 水生物种从边界水域或河道进入，模拟连通水体和上游输入
- 迁入前必须通过局部 habitat 检查
- 迁入后有冷却期，避免连续刷入

这套机制目前主要用于防止控制物种在随机波动后永久断链，例如：

- `fox`
- `shrimp`
- `blackfish`
- `pike`

### 陆地动物

典型逻辑：

- 植食动物根据植物总量决定产仔规模
- 肉食动物根据猎物总量决定产仔规模
- 天敌越多，产仔越谨慎
- 鸟类、杂食哺乳类和水边物种允许跨陆地 / 水域补充能量

### 水生生物

当前水生系统额外加入了软承载抑制，主要用于：

- `algae`
- `seaweed`
- `plankton`
- 水生自然繁殖生成入口

## 行为优先级

### 陆地动物

1. 逃跑 / 防御
2. 觅食
3. 求偶
4. 生产
5. 闲逛

其中觅食现在支持两类扩展：

- 替代植物源：例如浆果、蘑菇、蕨类、苔藓
- 跨生态位猎物：例如鸭、翠鸟、浣熊、貉会利用水边鱼虾和水面生物

### 防御行为

项目已接入的防御策略包括：

- 逃跑
- 反击
- 群体防御
- 伪装
- 物种专属防御（臭鼬喷雾、负鼠装死、犰狳蜷缩）

## 植物机制

植物当前包含：

- 生长
- 产种
- 发芽
- 季节性结果
- 轻量竞争

植物竞争现在包含：

- 空间竞争
- 光照竞争
- 根系竞争

## 环境机制

`Environment` 当前维护：

- 地形
- 水域
- 森林
- 沙地 / 岩石
- 水岸过渡带：泥地 / 沙地
- 时间：小时、天、季节
- 大气：天气、光照、风速、空气质量
- 土壤：湿度、肥沃度、有机质
- 水质：氧含量、清洁度、水温、酸碱度、流速、深度系数、营养盐

当前水域还细分为：

- `river_channel`
- `lake_shallow`
- `lake_deep`

这些差异会影响：

- 小鱼和狗鱼对河流高含氧环境的偏好
- 鲤鱼、河豚、虾对浅湖和高营养水体的偏好
- 鲶鱼、大鱼、黑鱼对深湖稳定环境的偏好
- 藻类和水草对高流速 / 高营养水体的不同响应
- 关键水生物种会主动向更适宜的河道、浅湖或深湖迁移，而不是仅靠随机游动
- 关键水生捕食者会优先沿局部高 prey 密度的水格移动，减少长期待在错误 habitat 中被动衰退
- 泥岸和浅滩会为底栖小型生物提供额外的碎屑食物和躲藏位，尤其影响虾这类浅水消费者
- 螃蟹等底栖捕食者在泥岸混浊区对虾的捕食成功率会下降
- 狗鱼会在河道、缓流浅滩和春季产卵带获得额外伏击与繁殖优势，更接近真实河道顶层捕食者行为
- 黑鱼和狗鱼在核心猎物过低时会降低高强度追猎，转向替代猎物并在冷季降低代谢，减少“清场后自灭”
- 黑鱼与狗鱼现在做了生态位分流：黑鱼更偏湖区鲤鱼，狗鱼更偏河道小鱼、蛙类和虾，减少顶层捕食者完全重叠
- `minnow` 作为河道饵鱼层承担了更多河流 prey 压力，并支持沿连通河道的自然补入

## 平衡监控

平衡系统当前是聚合式启发模型，不是逐物种硬编码理想数量。

它主要看 8 个面向：

- 陆地基础生产者
- 果类植物
- 授粉者
- 陆地中层消费者
- 陆地捕食者
- 水生基础生产者
- 水生中层消费者
- 水生捕食者

输出：

- 健康度
- 预警
- 调整建议
- 因果事件链

## 蝴蝶效应

`EcoBalance.analyze_cascade()` 会根据关键物种涨跌记录因果链。

例如：

- 兔子暴涨 -> 草可能下降
- 小鱼暴跌 -> 大鱼和黑鱼可能跟着下降
- 鲤鱼暴涨 -> 藻类与浮游生物可能承压

## 当前模型边界

- 水生行为比陆地行为更脚本化
- 两栖物种仍然混在陆地动物集合中
- 环境影响是启发式而非精确能量守恒
- 食谱已明显扩宽，但长期平衡仍然对参数和随机种子敏感
