# EcoWorld 物种手册

## 植物

| 物种 | 类型 | 关键特性 |
|---|---|---|
| `grass` | 基础植物 | 快速生长，风向播种 |
| `bush` | 灌木 | 提供庇护，传播种子 |
| `flower` | 花卉 | 依赖蜜蜂授粉 |
| `moss` | 苔藓 | 偏水边环境 |
| `tree` | 高大植物 | 提供遮蔽与庇护 |
| `vine` | 攀援植物 | 依赖支撑物 |
| `cactus` | 特殊植物 | 耐旱，有刺 |
| `berry` | 果实植物 | 吸引动物传播 |
| `mushroom` | 菌类 | 生长快，可能有毒 |
| `fern` | 阴湿植物 | 喜潮湿环境 |
| `apple_tree` | 果树 | 秋季结果 |
| `cherry_tree` | 果树 | 春季结果 |
| `grape_vine` | 果藤 | 夏季结果，依附支撑物 |
| `strawberry` | 果类 | 夏秋结果 |
| `blueberry` | 果类 | 夏季结果 |
| `orange_tree` | 果树 | 冬季结果 |
| `watermelon` | 果类 | 夏季结果 |

## 陆地动物

### 基础链路

| 物种 | 食性 | 关键作用 |
|---|---|---|
| `insect` | 植食 | 可取食草、花、苔藓和部分低矮浆果 |
| `rabbit` | 植食 | 可在草、花、灌木、苔藓和浆果间切换 |
| `mouse` | 杂食偏植 | 吃植物也吃昆虫、蜜蜂、蜘蛛 |
| `deer` | 植食 | 大型食草动物 |
| `fox` | 肉食 | 捕食兔、鼠、鸟、蛇，也会吃青蛙和鸭 |
| `wolf` | 肉食 | 顶级陆地捕食者，猎物结构更宽 |
| `snake` | 肉食 | 捕食昆虫、鼠类、兔子、麻雀和青蛙 |
| `spider` | 肉食 | 控制昆虫密度，也捕食蜜蜂和水黾 |
| `bee` | 植食 | 授粉核心物种，也会采食浆果花蜜 |

### 鸟类

| 物种 | 关键特性 |
|---|---|
| `bird` | 基础鸟类，可吃昆虫、鼠类、蜜蜂和水边小型生物 |
| `eagle` | 高位捕食者 |
| `owl` | 夜行捕食者，可覆盖陆地和水边小型猎物 |
| `duck` | 水陆两栖取食，可吃植物、小鱼、虾、浮游生物 |
| `swan` | 偏水域植食，也会过滤部分浮游生物 |
| `sparrow` | 小型杂食鸟，可吃草籽、浆果、昆虫、蜜蜂 |
| `parrot` | 偏果实取食 |
| `kingfisher` | 水边捕食鱼虾，也会吃蝌蚪、水黾和小型蛙类 |
| `magpie` | 杂食，兼吃浆果、小型节肢动物和蛙类 |
| `crow` | 杂食，可在陆地和水边机会取食 |
| `woodpecker` | 取食昆虫，也会抓蜜蜂和蜘蛛 |
| `hummingbird` | 花蜜 / 授粉，也会吸食浆果花蜜 |

### 哺乳 / 杂食

| 物种 | 关键特性 |
|---|---|
| `squirrel` | 攀爬、采食果实，也吃菌类和蕨类 |
| `hedgehog` | 小型防御性物种，可吃昆虫、蜘蛛、蜜蜂、蝌蚪 |
| `bat` | 夜行，捕食昆虫，也会吃蜘蛛、蜜蜂、水黾 |
| `raccoon` | 杂食，已接入水边机会捕食 |
| `bear` | 大型杂食，冬眠，能捕食鱼类、鸟类和两栖物种 |
| `wild_boar` | 掘食，会破坏植物 |
| `badger` | 夜行性，取食昆虫、小型脊椎动物和浆果 |
| `raccoon_dog` | 杂食，可涉水，可吃鱼虾蝌蚪 |
| `skunk` | 臭腺防御 |
| `opossum` | 装死 |
| `coati` | 杂食，善于攀爬，也吃鸟类、蜂类和蝌蚪 |
| `armadillo` | 装甲防御，也会吃蜘蛛、蝌蚪和水黾 |
| `frog` | 两栖，水里吃浮游生物，陆上吃昆虫、蜜蜂、蜘蛛 |

## 水生生物

| 物种 | 层级 | 关键特性 |
|---|---|---|
| `algae` | 生产者 | 高繁殖，基础能量来源 |
| `seaweed` | 生产者 | 相对稳定 |
| `plankton` | 生产者 | 水域基础食物 |
| `small_fish` | 消费者 | 以浮游生物为主，也会啃食藻类和表层小型生物 |
| `carp` | 消费者 | 杂食型鱼类，可在浮游生物、藻类、水草和小型无脊椎间切换 |
| `catfish` | 捕食者 | 捕食鱼虾，也会吃鲤鱼、青蛙和蝌蚪 |
| `large_fish` | 捕食者 | 高位鱼类 |
| `pufferfish` | 消费者 | 偏防御型，也会吃浮游生物和虾 |
| `shrimp` | 消费者 | 吃藻类，也会过滤浮游生物和啃食水草 |
| `crab` | 捕食者 | 捕食虾、小鱼、蝌蚪，也会机会取食浮游生物 |
| `tadpole` | 消费者 | 青蛙幼体，可吃藻类和浮游生物 |
| `water_strider` | 消费者 | 水面活动，可吃浮游生物、藻类碎屑和蝌蚪幼体 |
| `blackfish` | 捕食者 | 控制鲤鱼 |
| `pike` | 捕食者 | 控制鲤鱼和小鱼，也会吃虾、蝌蚪和青蛙 |

## 当前代码中的关键控制物种

- 陆地基础生产者：`grass`、`bush`、`flower`
- 授粉关键：`bee`、`hummingbird`
- 陆地主猎物：`rabbit`、`mouse`、`insect`
- 陆地高压捕食者：`fox`、`wolf`
- 水域基础生产者：`algae`、`plankton`
- 水域关键中层：`small_fish`、`carp`、`shrimp`
- 水域抑制捕食者：`blackfish`、`pike`、`large_fish`
