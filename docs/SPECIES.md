# EcoWorld 物种手册

当前版本共 **66 个物种**：

- 植物 `17`
- 陆地动物、鸟类与两栖 `34`
- 水生物种 `15`

## 当前关键控制物种

- 陆地基础生产者：`grass`、`bush`、`flower`
- 授粉关键：`bee`、`hummingbird`
- 陆地主猎物：`rabbit`、`mouse`、`insect`
- 陆地捕食控制：`fox`、`wolf`
- 水域基础生产者：`algae`、`plankton`
- 水域中层关键：`small_fish`、`minnow`、`carp`、`shrimp`
- 水域高位控制：`blackfish`、`pike`、`large_fish`、`catfish`
