# EcoWorld 物种手册

当前版本共 **67 个物种**：

- 植物 `17`
- 陆地动物、鸟类与两栖 `35`
- 水生物种 `15`

本文档按“代码名 / 中文名 / 生态位 / 食性与资源 / 栖息偏好 / 主要关系 / 关键特征”整理，尽量覆盖当前实现中的真实行为。

## 植物

### `grass` / 草

- 生态位：陆地基础生产者
- 食性与资源：为兔子、鹿、昆虫等提供持续食物
- 栖息偏好：开阔陆地，适应面最广
- 主要关系：被中小型草食动物高频消耗
- 关键特征：生长快、扩散快，是陆地食物网底层核心

### `bush` / 灌木

- 生态位：低矮结构植物
- 食性与资源：为兔子、鹿、松鼠、麻雀等提供食物和遮蔽
- 栖息偏好：林缘、灌丛带、开阔地边缘
- 主要关系：为灌丛位 `shrub_shelter` 提供资源
- 关键特征：既是食物也是庇护物，是小型鸟兽的重要栖位来源

### `flower` / 花

- 生态位：授粉链核心植物
- 食性与资源：为蜜蜂、蜂鸟提供花蜜
- 栖息偏好：开阔阳光地带
- 主要关系：支撑 `bee`、`hummingbird` 和结果植物链
- 关键特征：会拉动授粉者和花蜜位 `nectar_patch`

### `moss` / 苔藓

- 生态位：阴湿低矮植物
- 食性与资源：可被兔子等低位消费者利用
- 栖息偏好：水边、阴湿地、林下
- 主要关系：与湿地和水岸过渡环境耦合
- 关键特征：耐阴湿，强化水边微生态

### `tree` / 树

- 生态位：高大结构植物
- 食性与资源：为多类树栖、夜栖物种提供结构和遮蔽
- 栖息偏好：林地与林缘
- 主要关系：生成 `canopy_roost` 与 `night_roost`
- 关键特征：是树冠链、夜栖链和松鼠链的基础资源

### `vine` / 藤蔓

- 生态位：攀援结构植物
- 食性与资源：增加立体结构和局部遮蔽
- 栖息偏好：依附树木和灌木
- 主要关系：增强林缘复杂度
- 关键特征：提高结构多样性，但不是主要底层食物

### `cactus` / 仙人掌

- 生态位：耐旱特殊植物
- 食性与资源：提供少量特殊植物资源
- 栖息偏好：干燥地带
- 主要关系：对一般草食链贡献较小
- 关键特征：耐旱、带防御性，偏生态补充角色

### `berry` / 浆果灌木

- 生态位：果灌植物
- 食性与资源：为鸟类、小型哺乳动物和杂食动物提供果源
- 栖息偏好：林缘、灌丛带
- 主要关系：同时提供 `shrub_shelter` 和部分 `nectar_patch`
- 关键特征：兼顾食物和庇护，是灌丛链关键节点

### `mushroom` / 蘑菇

- 生态位：菌类资源
- 食性与资源：为松鼠等杂食/植食物种提供补充食物
- 栖息偏好：阴湿林下
- 主要关系：补充林下食物多样性
- 关键特征：寿命较短、恢复快

### `fern` / 蕨类

- 生态位：阴湿地表植物
- 食性与资源：被兔子、鹿、松鼠等低位消费者利用
- 栖息偏好：林下、湿地边缘
- 主要关系：增强阴湿地的连续植物资源
- 关键特征：连接林下与湿地边缘植物带

### `apple_tree` / 苹果树

- 生态位：大型果树
- 食性与资源：为鸟类、松鼠、杂食动物提供高能量果实
- 栖息偏好：树林和果树林带
- 主要关系：提供 `canopy_roost`、`night_roost`
- 关键特征：秋季果源，对树冠链和果食链都重要

### `cherry_tree` / 樱桃树

- 生态位：春季果树
- 食性与资源：早季果源和树冠结构
- 栖息偏好：林缘、果树林带
- 主要关系：支撑春季鸟类和树栖动物
- 关键特征：春季结果，帮助季节早期维持树冠链

### `grape_vine` / 葡萄藤

- 生态位：攀援果藤
- 食性与资源：提供夏季果源
- 栖息偏好：依附支撑物附近
- 主要关系：增强林缘果源多样性
- 关键特征：结果集中，偏季节性补给

### `strawberry` / 草莓

- 生态位：低矮果类
- 食性与资源：为小型鸟类和低位植食者提供近地果源
- 栖息偏好：开阔地、林缘低层
- 主要关系：支撑小型杂食链和花蜜链
- 关键特征：兼具果源与低位花蜜价值

### `blueberry` / 蓝莓

- 生态位：浆果灌木
- 食性与资源：鸟类、松鼠、杂食动物的稳定果源
- 栖息偏好：灌丛带和林缘
- 主要关系：提供灌丛位和果源
- 关键特征：对 `sparrow`、`hummingbird`、`squirrel` 都有支撑

### `orange_tree` / 橙子树

- 生态位：冬季果树
- 食性与资源：为冬季残存鸟兽提供果源
- 栖息偏好：林缘和果树林带
- 主要关系：提供树冠位与夜栖位
- 关键特征：冬季资源节点，帮助冷季维持部分树栖链

### `watermelon` / 西瓜

- 生态位：高能量地表果实
- 食性与资源：为地表杂食和植食者提供高热量补给
- 栖息偏好：开阔地
- 主要关系：更多影响地表食物多样性
- 关键特征：单体营养高，但不提供结构栖位

## 陆地动物、鸟类与两栖

### `insect` / 昆虫

- 生态位：陆地底层消费者
- 食性与资源：取食草、花、低矮植物资源
- 栖息偏好：草地、花丛、灌丛边
- 主要关系：被鸟类、蜘蛛、青蛙、蝙蝠大量捕食
- 关键特征：是空中链、树冠链和两栖链的底层能量来源

### `night_moth` / 夜飞蛾

- 生态位：夜行底层飞虫
- 食性与资源：花蜜、灌丛花源、湿地边缘有机资源
- 栖息偏好：花丛、湿地、灌丛、水边林缘
- 主要关系：被 `bat`、`owl`、`frog`、`spider`、`bird`、`kingfisher` 利用
- 关键特征：连接花源、湿地和夜行捕食链，是为夜行链新增的自然底层资源物种

### `rabbit` / 兔子

- 生态位：中层草食动物
- 食性与资源：草、花、灌木、苔藓、浆果、蕨类
- 栖息偏好：开阔地和灌丛边缘
- 主要关系：是 `fox`、`wolf`、`eagle` 等的主要猎物
- 关键特征：繁殖快，是陆地中层猎物核心

### `fox` / 狐狸

- 生态位：中高位陆地捕食者
- 食性与资源：兔子、老鼠、鸟类、蛇、青蛙、昆虫等
- 栖息偏好：林缘、草地、灌丛地
- 主要关系：控制兔鼠和部分中小型动物
- 关键特征：捕食会保留猎物安全线，避免完全清空

### `deer` / 鹿

- 生态位：大型草食动物
- 食性与资源：草、灌木、苔藓、花、浆果、蕨类
- 栖息偏好：林缘草地
- 主要关系：受 `wolf` 控制
- 关键特征：体量大，消耗植物资源较多

### `mouse` / 老鼠

- 生态位：小型底层猎物
- 食性与资源：植物、种子、昆虫、蜘蛛、蜜蜂等
- 栖息偏好：灌丛和林下
- 主要关系：是狐狸、蛇、猫头鹰等的猎物
- 关键特征：兼有植食和小型猎食行为

### `bird` / 小鸟

- 生态位：泛化小型鸟类
- 食性与资源：昆虫、小型节肢动物、部分植物资源
- 栖息偏好：林缘、灌丛、水边
- 主要关系：兼具捕食和被捕食角色
- 关键特征：是通用鸟类模板，生态位较宽

### `snake` / 蛇

- 生态位：陆地隐蔽捕食者
- 食性与资源：昆虫、老鼠、兔子、麻雀、青蛙等
- 栖息偏好：草丛、灌丛、林下
- 主要关系：压制小型陆地和两栖物种
- 关键特征：低速隐蔽、高杀伤，靠埋伏控制中小型猎物

### `bee` / 蜜蜂

- 生态位：授粉者
- 食性与资源：花蜜和花源资源
- 栖息偏好：花丛和灌丛边缘
- 主要关系：支撑果类植物和花源链
- 关键特征：是花蜜链与果实链稳定性的关键

### `frog` / 青蛙

- 生态位：两栖中层消费者
- 食性与资源：水中吃浮游生物、水黾和藻类，陆上吃昆虫、蜜蜂、蜘蛛
- 栖息偏好：湿地、水边、岸带过渡区
- 主要关系：连接水生底层和陆地昆虫链
- 关键特征：繁殖依赖 `wetland_patch / riparian_perch`，是两栖链核心

### `eagle` / 老鹰

- 生态位：高位空中捕食者
- 食性与资源：兔子、老鼠、蛇、鸟类、青蛙、小鱼等
- 栖息偏好：开阔上空与林缘
- 主要关系：压制中小型脊椎动物
- 关键特征：视野广、速度快、俯冲捕食强

### `owl` / 猫头鹰

- 生态位：夜行高位鸟类
- 食性与资源：昆虫、老鼠、蝙蝠、水黾、青蛙
- 栖息偏好：树冠、夜栖位、林缘
- 主要关系：夜间压制小型夜行和地表物种
- 关键特征：依赖 `night_roost / canopy_roost`，昼夜行为差异明显

### `duck` / 鸭子

- 生态位：水陆混合取食者
- 食性与资源：植物、小鱼、虾、浮游生物等
- 栖息偏好：水边和浅水区
- 主要关系：连接水陆取食链
- 关键特征：跨生态位觅食，资源面较宽

### `swan` / 天鹅

- 生态位：大型水鸟
- 食性与资源：水生植物和部分浮游资源
- 栖息偏好：浅湖和缓流水体
- 主要关系：影响水草和浅水生产者
- 关键特征：偏植食和过滤型取食

### `sparrow` / 麻雀

- 生态位：小型杂食鸟
- 食性与资源：草籽、浆果、昆虫、蜜蜂
- 栖息偏好：灌丛和林缘
- 主要关系：依赖灌丛位和低层食物
- 关键特征：小型、脆弱，但灌丛链恢复后可持续存在

### `parrot` / 鹦鹉

- 生态位：果食鸟类
- 食性与资源：果实和植物资源
- 栖息偏好：树木和果树林带
- 主要关系：更依赖果树和树冠
- 关键特征：偏果食，结构上更接近树冠链

### `kingfisher` / 翠鸟

- 生态位：近水捕食鸟
- 食性与资源：小鱼、米诺鱼、虾、蝌蚪、水黾、青蛙
- 栖息偏好：岸带、灌丛、水边树枝
- 主要关系：依赖 `riparian_perch / shrub_shelter`
- 关键特征：沿岸俯冲捕食，是水边食物链上层关键物种

### `wolf` / 狼

- 生态位：陆地顶层捕食者
- 食性与资源：鹿、兔子、老鼠等
- 栖息偏好：林缘和开阔地
- 主要关系：控制大型和中型草食动物
- 关键特征：承担陆地高位控制任务

### `spider` / 蜘蛛

- 生态位：小型节肢捕食者
- 食性与资源：昆虫、蜜蜂等
- 栖息偏好：灌丛、草丛、林缘
- 主要关系：压制昆虫基底
- 关键特征：是防止昆虫爆量的关键控制者

### `magpie` / 喜鹊

- 生态位：中型杂食鸟
- 食性与资源：浆果、小型节肢动物、青蛙等
- 栖息偏好：林缘和水边
- 主要关系：兼具果食和机会捕食
- 关键特征：生态位较宽，能跨陆地与水边补食

### `crow` / 乌鸦

- 生态位：泛食性中型鸟
- 食性与资源：机会型摄食，植物与动物资源都能利用
- 栖息偏好：林缘、开阔地、水边
- 主要关系：更多是机会取食者而非强控制者
- 关键特征：适应面广，但竞争优势不如鹰或狐狸

### `woodpecker` / 啄木鸟

- 生态位：树干昆虫捕食者
- 食性与资源：昆虫、蜜蜂、蜘蛛
- 栖息偏好：树木密集区
- 主要关系：依赖树木结构
- 关键特征：更贴近树冠和树干昆虫链

### `hummingbird` / 蜂鸟

- 生态位：花蜜型小型鸟
- 食性与资源：花蜜、浆果花源、小昆虫
- 栖息偏好：灌丛、花丛、林缘低层
- 主要关系：依赖 `nectar_patch / shrub_shelter`
- 关键特征：连接授粉链和小型空中昆虫链

### `squirrel` / 松鼠

- 生态位：树栖小型哺乳动物
- 食性与资源：果实、菌类、蕨类、部分植物资源
- 栖息偏好：树冠和树木密集区
- 主要关系：依赖 `canopy_roost`
- 关键特征：树冠链代表物种，资源不足时易快速衰退

### `hedgehog` / 刺猬

- 生态位：地表小型杂食动物
- 食性与资源：昆虫、蜘蛛、蜜蜂、蝌蚪等
- 栖息偏好：灌丛和林下
- 主要关系：控制地表小型无脊椎动物
- 关键特征：具防御性，属于小型抗捕食物种

### `bat` / 蝙蝠

- 生态位：夜行空中捕食者
- 食性与资源：昆虫、蜘蛛、蜜蜂、水黾
- 栖息偏好：夜栖位、树冠、林带
- 主要关系：依赖 `night_roost / canopy_roost`
- 关键特征：夜间活动强，白天会回栖位恢复

### `raccoon` / 浣熊

- 生态位：水边杂食哺乳动物
- 食性与资源：植物、果实、小型水生猎物
- 栖息偏好：水边林带和灌丛
- 主要关系：跨陆地与水边取食
- 关键特征：水边机会型捕食者

### `bear` / 熊

- 生态位：大型杂食顶层动物
- 食性与资源：果实、鱼类、鸟类、两栖、哺乳动物
- 栖息偏好：林地与水边
- 主要关系：生态位很宽，对多条链有干扰能力
- 关键特征：高支配力、高能量摄取

### `wild_boar` / 野猪

- 生态位：大型掘食杂食动物
- 食性与资源：地表植物和杂食资源
- 栖息偏好：林缘和灌丛
- 主要关系：影响低层植物结构
- 关键特征：会扰动植物层和地表资源

### `badger` / 獾

- 生态位：中小型夜行杂食动物
- 食性与资源：昆虫、小型脊椎动物、浆果
- 栖息偏好：林下和灌丛
- 主要关系：夜间参与地表猎食
- 关键特征：兼具夜行和杂食特征

### `raccoon_dog` / 貉

- 生态位：杂食夜行哺乳动物
- 食性与资源：植物、鱼虾、蝌蚪等
- 栖息偏好：水边灌丛和林缘
- 主要关系：跨陆水补食
- 关键特征：资源面宽，偏机会取食

### `skunk` / 臭鼬

- 生态位：小型杂食防御型动物
- 食性与资源：地表小型资源和植物资源
- 栖息偏好：灌丛和草地边缘
- 主要关系：更多依靠防御而非速度
- 关键特征：臭腺防御，降低被持续捕食概率

### `opossum` / 负鼠

- 生态位：小型夜行杂食动物
- 食性与资源：植物、小型动物、腐食型机会资源
- 栖息偏好：林缘和灌丛
- 主要关系：偏机会存活者
- 关键特征：装死防御

### `coati` / 长鼻浣熊

- 生态位：攀爬型杂食动物
- 食性与资源：果实、鸟类、蜂类、蝌蚪等
- 栖息偏好：树木和水边林带
- 主要关系：跨树冠和地表两层
- 关键特征：兼具攀爬和杂食

### `armadillo` / 犰狳

- 生态位：地表防御型杂食动物
- 食性与资源：蜘蛛、蝌蚪、水黾等小型猎物
- 栖息偏好：地表和灌丛地带
- 主要关系：偏底层小型猎食
- 关键特征：装甲防御，降低高位捕食成功率

## 水生物种

### `algae` / 藻类

- 生态位：水域基础生产者
- 食性与资源：为虾、小鱼、鲤鱼等提供底层能量
- 栖息偏好：高营养浅水和静水区
- 主要关系：被虾、小型鱼类持续利用
- 关键特征：增长快，是水域底层能量核心

### `seaweed` / 水草

- 生态位：结构型水生植物
- 食性与资源：提供更稳定的水生植物资源
- 栖息偏好：浅水和缓流水体
- 主要关系：被鲤鱼、虾等利用
- 关键特征：比藻类更稳定，也会影响局部结构

### `plankton` / 浮游生物

- 生态位：水域底层食物
- 食性与资源：支撑小鱼、米诺鱼、虾、蝌蚪
- 栖息偏好：各类水体，尤其高光高营养区
- 主要关系：是中层鱼和两栖幼体的底层食物
- 关键特征：扩张快，对水域稳定性影响极大

### `small_fish` / 小鱼

- 生态位：水域中层消费者
- 食性与资源：浮游生物、藻类、水面小型资源
- 栖息偏好：高氧河道和浅湖
- 主要关系：被鲶鱼、大鱼、黑鱼、狗鱼、翠鸟等捕食
- 关键特征：是水域中层通用 prey

### `minnow` / 米诺鱼

- 生态位：河道中层小型鱼
- 食性与资源：浮游生物、藻类、水面小型资源
- 栖息偏好：河道、浅滩、浅湖边缘
- 主要关系：是 `pike` 和部分水边链的关键 prey
- 关键特征：承担河道 prey 缓冲层，维持水边链稳定

### `carp` / 鲤鱼

- 生态位：湖区中层杂食鱼
- 食性与资源：浮游生物、藻类、水草、小型无脊椎资源
- 栖息偏好：低流速、高营养湖区
- 主要关系：被黑鱼、大鱼、狗鱼捕食
- 关键特征：偏湖区资源利用者，容易在适宜水体占优

### `catfish` / 鲶鱼

- 生态位：底栖高位鱼
- 食性与资源：虾、鲤鱼、青蛙、蝌蚪，条件合适时也捕食小鱼
- 栖息偏好：河道和部分深水区
- 主要关系：更偏底栖 prey，而不是专门追河道饵鱼
- 关键特征：底栖压力大，能影响青蛙和虾链

### `large_fish` / 大鱼

- 生态位：湖区高位捕食鱼
- 食性与资源：鲤鱼、小鱼、虾、青蛙、蝌蚪
- 栖息偏好：深湖和稳定湖区
- 主要关系：与黑鱼一起压制湖区中层
- 关键特征：偏湖区顶层控制

### `pufferfish` / 河豚

- 生态位：防御型水生消费者
- 食性与资源：浮游生物、藻类、虾等
- 栖息偏好：浅湖和部分河道
- 主要关系：更多是防御型中层物种
- 关键特征：具防御特性，不是高位控制者

### `shrimp` / 虾

- 生态位：底栖中层消费者
- 食性与资源：藻类、浮游生物、水草、底栖碎屑
- 栖息偏好：浅湖、泥岸、缓流高营养区
- 主要关系：被鲶鱼、螃蟹、大鱼、黑鱼、狗鱼捕食
- 关键特征：受 `wetland_patch`、泥岸和底栖环境影响明显

### `crab` / 螃蟹

- 生态位：底栖捕食者
- 食性与资源：虾、小鱼、蝌蚪、浮游生物
- 栖息偏好：浅水和底栖环境
- 主要关系：压制虾和两栖幼体
- 关键特征：底栖捕食者，对浅湖底层结构影响大

### `tadpole` / 蝌蚪

- 生态位：两栖幼体
- 食性与资源：藻类、浮游生物
- 栖息偏好：浅水和湿地区域
- 主要关系：会变态成青蛙，是两栖链前段
- 关键特征：连接水域底层和青蛙种群恢复

### `water_strider` / 水黾

- 生态位：水面小型消费者
- 食性与资源：浮游生物、藻类碎屑、表层资源
- 栖息偏好：浅水、水边、水面带
- 主要关系：被小鱼、米诺鱼、青蛙、翠鸟等利用
- 关键特征：是水边和表层链的重要补充猎物

### `blackfish` / 黑鱼

- 生态位：湖区高位捕食鱼
- 食性与资源：以鲤鱼和湖区猎物为主
- 栖息偏好：深湖和稳定湖区
- 主要关系：和狗鱼生态位分流，更多压湖区
- 关键特征：湖区顶层控制者

### `pike` / 狗鱼

- 生态位：河道高位捕食鱼
- 食性与资源：米诺鱼、小鱼、虾、青蛙、蝌蚪
- 栖息偏好：河道、浅滩、高氧浅水带
- 主要关系：与 `minnow` 强耦合
- 关键特征：伏击型顶层鱼，是河道链稳定性的关键控制者

## 当前最关键的控制与脆弱物种

### 控制物种

- 陆地捕食控制：`fox`、`wolf`
- 授粉控制：`bee`、`hummingbird`

- 水边链：`kingfisher`
- 两栖链：`frog`
- 河道饵鱼层：`minnow`
