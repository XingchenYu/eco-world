## 一、竞争机制

### 1.1 食物竞争

**机制**：同种/异种动物争夺有限食物资源

**实现逻辑**：
```python
# 竞争系数
同种竞争系数 = 1.5  # 更激烈
异种竞争系数 = 1.0

# 食物可用性
食物可用性 = max(0.1, 1.0 - 竞争者数量 × 0.15)

# 竞争判定
得分 = 支配力 × 0.4 + 健康/100 × 0.3 + 性别加成 × 0.3
```

**影响**：
- 竞争失败者：减少觅食机会，可能饥饿
- 竞争激烈时：食物因子下降 → 繁殖率降低

### 1.2 领地竞争

**机制**：同种动物争夺领地

**实现逻辑**：
```python
# 领地范围
territory_range = 3  # 格

# 防守者优势
defender_bonus = 1.3

# 争夺判定
得分 = 支配力 × 防守加成 + 健康/100
```

**影响**：
- 领地被入侵：可能受伤（健康-5~15）
- 领地质量下降：繁殖受影响

### 1.3 配偶竞争

**机制**：多个雄性争夺一个雌性

**实现逻辑**：
```python
# 竞争得分
得分 = 支配力 × 0.5 + 健康/100 × 0.3 + 随机 × 0.2

# 失败惩罚
20%概率受伤（健康-3~10）
```

**影响**：
- 弱势雄性：失去繁殖机会，可能受伤
- 强势雄性：获得交配权

## 二、防御机制

### 2.1 逃跑行为

**触发条件**：发现捕食者

**成功率因素**：
- 速度差异（更快 = 更容易逃跑）
- 距离（更远 = 更容易逃跑）
- 环境遮蔽（草丛/灌木 = 更容易逃跑）

**公式**：
```python
逃跑成功率 = 0.3 + 速度优势 × 0.3 + 距离优势 × 0.3 + 遮蔽加成 × 0.1
```

### 2.2 反击行为

**适用动物**：刺猬、豪猪等有防御武器的动物

**触发条件**：
```python
can_counter_attack = True
counter_strength = 0.3  # 反击强度
```

**效果**：
- 成功反击：捕食者受伤（健康-5~15）
- 可能吓退捕食者

### 2.3 伪装行为

**适用动物**：变色龙、某些昆虫

**触发条件**：
```python
has_camouflage = True
camouflage_skill = 0.3  # 伪装技巧
```

**环境加成**：
- 在特定地形上伪装效果更好
- 如：绿色昆虫在草地上 +60% 成功率

### 2.4 群体防御

**适用动物**：羊群、鱼群等群居动物

**触发条件**：
- 至少3个同伴在场
- forms_groups = True

**公式**：
```python
防御成功率 = min(0.8, 0.2 + 群体大小 × 0.1)
```

**效果**：
- 成功防御：捕食者被吓跑
- 群体成员互助

### 2.5 装甲防御

**适用动物**：乌龟、穿山甲、刺猬

**触发条件**：
```python
defense_rating = 2.0  # 防御力（高于普通）
has_armor = True
```

**效果**：
- 减少被捕食概率
- 可能伤害捕食者（刺猬）

## 三、新增植物种类

### 3.1 大树 🌲
- **特点**：最高大，提供庇护所，生长最慢
- **寿命**：300 tick
- **生长率**：0.005（最慢）
- **防御**：提供树荫（影响周围植物生长）
- **竞争**：耐阴性强，根系深

### 3.2 藤蔓 🌿
- **特点**：攀援植物，需要支撑
- **寿命**：60 tick
- **生长率**：0.05（快）
- **依赖**：需要树/灌木支撑才能生长良好
- **竞争**：缠绕其他植物

### 3.3 仙人掌 🌵
- **特点**：耐旱，有刺防御
- **寿命**：120 tick
- **生长率**：0.01
- **防御**：30%概率刺伤捕食者
- **耐旱**：储存水分（50单位）

### 3.4 浆果灌木 🫐
- **特点**：果实吸引动物传播种子
- **寿命**：80 tick
- **生长率**：0.02
- **营养**：50（高）
- **互惠**：动物吃浆果时帮助传播种子

### 3.5 蘑菇 🍄
- **特点**：生长快，寿命短
- **寿命**：20 tick（最短）
- **生长率**：0.1（最快）
- **毒性**：30%概率有毒
- **环境**：在树荫下生长更好

### 3.6 蕨类 🌿
- **特点**：古老植物，喜阴湿
- **寿命**：100 tick
- **生长率**：0.025
- **繁殖**：孢子繁殖
- **环境**：雨天生长更快

## 四、植物竞争机制

### 4.1 光竞争
**机制**：高植物遮挡低植物

**公式**：
```python
遮挡因子 = 高植物大小 / 低植物大小
if 遮挡因子 > 1.5:
    低植物.生长率 *= 0.7
    低植物.健康 -= 1
```

### 4.2 水分竞争
**机制**：根系争夺水分

**判定**：
```python
if 根系范围重叠:
    较小植物.健康 -= 1
```

### 4.3 空间竞争
**机制**：密度影响生长

**判定**：
```python
if 周围植物数 > 5:
    植物.生长率 *= 0.8
```
