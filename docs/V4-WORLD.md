# V4 世界设计

## 1. 目标

`v4.0` 的世界不再是单地图生态沙盒，而是一个由多个生态区组成的多尺度行星生态网络。

它必须支持：

- 多大陆、多海盆、多气候带
- 陆地、淡水、海洋、天空四大生态域
- 多尺度模拟
- 区域间迁徙、扩散、回流和局部灭绝
- 玩家在全球、区域、局部三个视角下观察和干预生态系统

## 2. 世界结构

世界分为 5 层：

1. 行星层
2. 大区层
3. 生态区层
4. 地块层
5. 微栖位层

### 行星层

负责：

- 纬度带
- 风带
- 洋流
- 降水分布
- 海平面与冰盖
- 全球气候波动
- 迁徙主通道

### 大区层

负责：

- 大陆与群岛
- 山脉与高原
- 河流流域
- 海岸线与海盆
- 主要生态带

### 生态区层

负责：

- 温带森林
- 草原
- 热带雨林
- 湿地
- 红树林
- 珊瑚礁
- 外洋等区域的完整生态结构

### 地块层

负责：

- 土壤
- 植被
- 水文
- 开阔度
- 深度
- 遮蔽
- 风险热区

### 微栖位层

负责：

- 树冠
- 树洞
- 灌丛
- 花蜜带
- 倒木
- 巢穴
- 岸带浅滩
- 芦苇带
- 礁缝
- 清洁站
- 夜间飞虫云团等

## 3. 三层地图

### 全球图

建议尺寸：

- `1024x512` 起步
- 目标可扩到 `2048x1024`

负责：

- 全球地理结构
- 区域索引
- 大尺度迁徙
- 区域连通
- 全球气候

### 区域图

建议尺寸：

- `256x256` 到 `512x512`

负责：

- 单区域生态运行
- 区域内生物群系结构
- 区域种群与资源变化

### 焦点图

建议尺寸：

- `128x128` 到 `256x256`

负责：

- 个体行为
- 局部群体行为
- 捕食、领地、繁殖、共生、冲突
- 玩家主要观察和操作

## 4. 首批区域

首批固定 6 区：

1. 温带森林区
2. 温带草原区
3. 热带雨林与大河区
4. 湿地与湖泊区
5. 海岸与浅海区
6. 珊瑚礁海区

## 5. 区域连接

连接类型包括：

- `land_corridor`
- `river_network`
- `coastal_exchange`
- `ocean_current`
- `air_migration_lane`
- `island_isolation`

这些连接决定：

- 哪些物种可迁移
- 迁移成本
- 迁移季节
- 疾病/污染传播
- 资源脉冲跨区扩散

## 6. 世界运行模式

- 全球图低频运行
- 区域图中频运行
- 焦点图高频运行

玩家焦点区域优先高精度模拟，非焦点区域保持生态连续，但采用聚合与降频策略。

- `canopy_niche`
- `tree_hollow`
- `shrub_shelter`
- `flower_patch`
- `fallen_log`
- `burrow_zone`
- `fungal_bed`

淡水：

- `reed_belt`
- `shore_hatch`
- `mud_bank`
- `riparian_perch`
- `shallow_spawn_bed`

海岸与海洋：

- `tide_pool`
- `mangrove_root`
- `seagrass_refuge`
- `reef_crevice`
- `cleaning_station`
- `deep_vent`

天空：

- `thermal_column`
- `night_swarm`
- `migration_rest_lane`
