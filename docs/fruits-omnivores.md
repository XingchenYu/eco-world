## 一、果类植物（7种）

### 1.1 🍎 苹果树 (AppleTree)
- **果实季节**：秋季
- **寿命**：200 tick
- **营养**：100（高）
- **特点**：
  - 大型果树，提供庇护所
  - 每棵树结果 10-25 个
  - 果实营养最高

### 1.2 🍒 樱桃树 (CherryTree)
- **果实季节**：春季
- **寿命**：150 tick
- **营养**：70
- **特点**：
  - 春季开花结果
  - 每棵树结果 15-40 个
  - 花期美丽

### 1.3 🍇 葡萄藤 (GrapeVine)
- **果实季节**：夏季
- **寿命**：80 tick
- **营养**：60
- **特点**：
  - 攀援果树，需要支撑
  - 有支撑时生长更快
  - 每株 5-15 串

### 1.4 🍓 草莓 (Strawberry)
- **果实季节**：夏秋季
- **寿命**：30 tick
- **营养**：40
- **特点**：
  - 地面小型果实
  - 繁殖快
  - 每株 3-10 个

### 1.5 🫐 蓝莓 (Blueberry)
- **果实季节**：夏季
- **寿命**：60 tick
- **营养**：50
- **特点**：
  - 小型浆果灌木
  - 每株 8-20 个
  - 营养丰富

### 1.6 🍊 橙子树 (OrangeTree)
- **果实季节**：冬季
- **寿命**：180 tick
- **营养**：80
- **特点**：
  - 冬季果实
  - 提供庇护所
  - 每棵树 8-18 个

### 1.7 🍉 西瓜 (Watermelon)
- **果实季节**：夏季
- **寿命**：40 tick
- **营养**：120（最高）
- **特点**：
  - 大型果实
  - 数量少但营养极高
  - 每株 1-3 个

## 二、杂食动物（8种）

### 2.1 🐻 熊 (Bear)
- **体型**：大型
- **寿命**：120 tick
- **食性**：杂食（果实 + 动物）
- **特点**：
  - 高支配力（1.5）
  - 冬季冬眠
  - 无天敌
  - 吃各种果实和小动物

### 2.2 🐗 野猪 (WildBoar)
- **体型**：中型
- **寿命**：70 tick
- **食性**：杂食
- **特点**：
  - 用鼻子掘食
  - 可能伤害植物根系
  - 天敌：狼、熊

### 2.3 🦡 獾 (Badger)
- **体型**：小型
- **寿命**：50 tick
- **食性**：杂食
- **特点**：
  - 夜行性
  - 有洞穴
  - 冬季活动减少

### 2.4 🐕 貉 (RaccoonDog)
- **体型**：小型
- **寿命**：45 tick
- **食性**：杂食
- **特点**：
  - 像浣熊的犬科
  - 能游泳
  - 天敌：狼、熊、狐狸

### 2.5 🦨 臭鼬 (Skunk)
- **体型**：小型
- **寿命**：35 tick
- **食性**：杂食
- **特点**：
  - 臭腺防御（70%成功率）
  - 很少有捕食者
  - 高防御力

### 2.6 🐭 负鼠 (Opossum)
- **体型**：小型
- **寿命**：25 tick
- **食性**：杂食
- **特点**：
  - 会装死（60%成功率）
  - 繁殖快
  - 天敌：狼、狐狸、猫头鹰

### 2.7 🦝 长鼻浣熊 (Coati)
- **体型**：小型
- **寿命**：40 tick
- **食性**：杂食
- **特点**：
  - 热带杂食动物
  - 会爬树
  - 喜欢果实

### 2.8 🦔 犰狳 (Armadillo)
- **体型**：小型
- **寿命**：50 tick
- **食性**：杂食
- **特点**：
  - 装甲外壳（防御力 2.5）
  - 蜷缩成球防御（80%成功率）
  - 无天敌
