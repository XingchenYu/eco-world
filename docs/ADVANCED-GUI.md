# EcoWorld 高级界面指南

高级界面文件位于 [src/renderer/advanced_gui.py](../src/renderer/advanced_gui.py)。

## 启动

```bash
PYTHONPATH=. python3 src/main.py --advanced
```

## 界面组成

- 主地图区域
- 右侧侧边栏
- 小地图
- 底部状态栏
- 天气粒子效果

## 功能

- 缩放
- 平移
- 窗口尺寸变化自适应
- 相机边界约束
- 选中生物
- 快速投放物种
- 物种统计面板
- 食物链面板
- 事件日志面板
- 设置面板
- 天气粒子与小地图视野框

## 快捷键

| 按键 | 功能 |
|---|---|
| `Space` | 暂停 / 继续 |
| `+` / `=` | 加速 |
| `-` | 减速 |
| `G` | 显示 / 隐藏网格 |
| `F1` | 物种统计 |
| `F2` | 食物链面板 |
| `F3` | 事件日志 |
| `F4` | 设置面板 |
| `Home` | 重置视角 |
| `Q` / `Esc` | 退出 |

## 快速生成

| 按键 | 物种 |
|---|---|
| `1` | `grass` |
| `2` | `rabbit` |
| `3` | `fox` |
| `4` | `wolf` |
| `5` | `deer` |
| `6` | `bear` |
| `7` | `apple_tree` |
| `8` | `carp` |
| `9` | `blackfish` |

## 鼠标操作

| 操作 | 说明 |
|---|---|
| 左键点击 | 选中生物或植物 |
| 右键拖拽 | 平移地图 |
| 滚轮 | 缩放视图 |
| 调整窗口大小 | 自动重排游戏区、侧栏和小地图 |

## 选中信息

当前选中对象会高亮，并在侧栏中显示基础信息，如：

- 物种
- 位置
- 年龄
- 健康
- 饥饿
- 当前行为状态

## 当前版本的界面修复

- 修复窗口缩放后侧边栏和游戏区尺寸错位
- 修复相机可拖出世界边界导致空白区域
- 修复事件日志面板读取事件对象时的崩溃
- 修复天气粒子仍使用旧天气名的问题
- 修复经典界面地形纹理每帧随机导致的闪烁

## 适用场景

- 观察复杂群落演化
- 查看某一物种的分布
- 手动做干预实验
- 查看预警、事件和健康度变化
