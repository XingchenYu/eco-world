## 微栖位可视化

当前界面已支持微栖位叠层，默认开启。

可显示的资源类型：

- `canopy_roost` 树冠位
- `night_roost` 夜栖位
- `shrub_shelter` 灌丛位
- `nectar_patch` 花蜜位
- `wetland_patch` 湿地位
- `riparian_perch` 岸栖位

设置面板会显示每类 patch 的：

- 数量
- 占用量

选中生物时，还会显示：

- 中文物种名
- 位置
- 生命
- 饥饿
- 年龄
- 繁殖资源类型
- 局部 patch 可用度
