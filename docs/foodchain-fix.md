# 2026-04-05 EcoWorld 食物链修复

## 修复原则
用户反馈：不能靠硬编码上限控制种群数量，应该引入天敌自然控制。

## 核心改进

### 1. 移除所有硬编码上限
**之前**：各种物种有硬编码上限（如鲤鱼上限100，小鱼上限60）
**现在**：所有繁殖逻辑基于食物充足度 + 天敌压力动态调整

| 物种 | 之前上限 | 现在控制方式 |
|-----|---------|-------------|
| 鲤鱼 | current_carp < 100 | 天敌（黑鱼/狗鱼）+ 食物因子 |
| 小鱼 | current_smallfish < 60 | 天敌压力 + 食物因子 |
| 藻类 | current_algae < 120 | 捕食压力补偿机制 |
| 水草 | current_seaweed < 80 | 捕食压力补偿机制 |
| 浮游生物 | current_plankton < 150 | 捕食压力补偿机制 |
| 虾 | current_shrimp < 80 | 天敌压力 + 食物因子 |
| 兔子 | current_rabbit < 100 | 天敌压力 + 食物因子 |
| 狐狸 | current_fox < 15 | 猎物充足度 |
| 昆虫 | current < 200 | 天敌压力 + 食物因子 |

### 2. 新增天敌物种

#### 水生食物链
| 天敌 | 控制目标 | 机制 |
|-----|---------|-----|
| **黑鱼 (Blackfish)** | 鲤鱼 | 专门捕食鲤鱼，繁殖率由鲤鱼数量决定 |
| **狗鱼 (Pike)** | 鲤鱼/小鱼 | 游速快，追击范围大，繁殖率由猎物数量决定 |

#### 陆地食物链
| 天敌 | 控制目标 | 机制 |
|-----|---------|-----|
| **狼 (Wolf)** | 鹿/兔子/老鼠 | 顶级捕食者，繁殖率由猎物数量决定 |
| **蜘蛛 (Spider)** | 昆虫 | 控制昆虫数量，繁殖率由昆虫数量决定 |

### 3. 完善食物链关系

**水生生物**：
- 藻类 → 被虾、浮游生物、鲤鱼捕食
- 浮游生物 → 被小鱼、虾、蝌蚪捕食
- 小鱼 → 被鲶鱼、大鱼、黑鱼、狗鱼、螃蟹、翠鸟捕食
- 鲤鱼 → 被黑鱼、大鱼、狗鱼捕食
- 虾 → 被鲶鱼、螃蟹、大鱼、黑鱼捕食

**陆地动物**：
- 昆虫 → 被狐狸、鸟、蛇、麻雀、猫头鹰、青蛙、蜘蛛捕食
- 兔子 → 被狐狸、狼、蛇、老鹰捕食
- 鹿 → 被狼捕食（之前无天敌导致泛滥）
- 老鼠 → 被狐狸、蛇、鸟捕食
- 小鸟 → 狐狸、老鹰捕食

### 4. 动态繁殖机制

所有物种繁殖遵循公式：
```
有效繁殖率 = 基础繁殖率 × 食物因子 × 天敌压力因子
```

- **食物因子**：猎物数量 / 捕食者数量 × 理想比例
- **天敌压力因子**：max(0.2, 1.0 - 天敌数量 × 0.05~0.1)

**补偿机制**：
- 被捕食越多 → 繁殖越积极（如藻类、浮游生物）
- 天敌压力越大 → 繁殖越谨慎（如小鱼、兔子）
